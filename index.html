Bug report #0001 [Interface overlapping]

LABELS

Category: Graphic elements

Priority: Extreme

Severity: Minor

Description

Priority: Extreme

Severity: Minor

Description: When the user selects option 1 (terminate process), after confirming the choice, the text in the console shifts by an indeterminate number of lines, overlapping the information panel in the console.

Expected result: Separated presentation of the navigation and information blocks without overlapping.

Actual result: The text from the information panel overlaps the text from the navigation block.

Steps:

  1. Open a cmd console

  2. Run program "OS_Lab3.exe" with additional parameters "10 100 50000 6 -f"

  3. Choose option 1 (kill process)

  4. Choose one of the processes to kill

  5. Confirm your choice by pressing "Enter"

Additional information:

  • Current build: 2.34

  • OS: Windows 10
